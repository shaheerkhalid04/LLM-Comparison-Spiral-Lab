# LLM Comparison ‚Äì Document Summarization

Compare how different Large Language Models (LLMs) summarize the same document
using the same prompt, then evaluate the results.

## What this project does

1. Reads one source document.
2. Sends the **same prompt** to three models (GPT, Gemini, Claude).
3. Saves each model's summary to a separate file.
4. Creates a comparison table and a report so you can score and conclude.

## Setup

1. (Recommended) create a virtual environment:

   ```
   python -m venv venv
   source venv/bin/activate      # on Windows: venv\Scripts\activate
   ```

2. Install the libraries:

   ```
   pip install -r requirements.txt
   ```

3. Add your API keys:

   ```
   cp .env.example .env
   ```

   Open `.env` and paste in your real keys. You need at least the keys for
   the models you want to run. The script skips any model whose key is
   missing and reports it, so you can start with just one provider if you
   like.

4. Load the keys into your shell (simple option):

   ```
   export $(cat .env | xargs)
   ```

## Running

1. Put your article, paper, or PDF text into `documents/source_document.txt`
   (a sample is already there).

2. Run:

   ```
   python src/run_comparison.py
   ```

3. Read the summaries in `outputs/`, then fill in the scores in
   `results/comparison_table.md` and write your conclusion in
   `results/report.md`.

## Evaluation metrics

Each model is scored 1 (poor) to 5 (excellent) on:

- **Summary Quality** ‚Äì is it well-structured and easy to understand?
- **Accuracy** ‚Äì does it correctly represent the source?
- **Conciseness** ‚Äì does it drop fluff while keeping key information?
- **Hallucinations** ‚Äì does it avoid adding facts not in the source?
  (higher score = fewer made-up facts)
